## @coolbuttons/vanilla

### Option 1: CDN 

Add this single line to your HTML `<head>`:

```html
<script src="https://cdn.jsdelivr.net/npm/@coolbuttons/vanilla@1.0.6/dist/coolbuttons.min.js"></script>
```

### Option 2: NPM (For Projects)

Install via npm:

```bash
npm install @coolbuttons/vanilla
```

Import in your JavaScript:

```javascript
import { createButton } from '@coolbuttons/vanilla';
```
